Government Organisations

Check a request from the Government Cyber Coordination Centre (GC3)

If you have recieved a vulnerability notification from the GC3, it should have come from the email vm@gc3.security.gov.uk with a "GC3-####" reference. To verify a communication or if you have any other questions, submit a request via the my-gc3 portal or send a new email (do not reply) to vm@gc3.security.gov.uk.

How to add vulnerability disclose programme details

If you are a government organisation you can enroll your vulnerability disclose programme or contact details by creating an issue or pull request against the co-cddo/gccc-vrs-iac GitHub repository.

How to implement the security.txt

You can configure a redirect to the central security.txt in a number of ways. See the co-cddo/implement-security.txt GitHub repository for guidance and code.
